# SCAPES: Scaling Lassa Virus Dynamics within Anthropogenic Ecosystems
**Master Repository Dictionary & Open Science Hub**

This repository serves as the canonical open-science hub and linkout dictionary for the **SCAPES** project. To comply with funder open-science mandates (NIH / NSF / NIFA / BBSRC) and adhere to FAIR (Findable, Accessible, Interoperable, and Reusable) data principles, all non-identifiable spatial covariates, analytical pipelines, and Quarto modeling scripts are indexed below. 

> **Data Governance & Privacy Notice:** To protect participant anonymity and comply with ethical approvals (NHREC/01/01/2007-17/07/2023; PSU STUDY00019989), raw high-resolution GPS tracking tracks, precise household geolocations, and human serological linkages are maintained in a secure, private consortium repository. In all public data releases indexed below, human spatial tracks are projected into a local null-space via an affine transformation, and human serological outcomes are reported strictly as aggregated municipal summaries.

### Supplementary Protocol Library (SOPs)
For full technical reproducibility, detailed Standard Operating Procedures utilized across the SCAPES data streams are archived within this repository's open documentation:    

* **`sops/UAV_flight_and_processing_protocol.pdf`**: Granular flight parameters and CVAT segmentation workflows.
* **`sops/rodent_trapping_and_biosafety_protocol.pdf`**: 10x10 agricultural grid spacing, peridomestic household Sherman/Tomahawk deployment, exploratory camera trap calibration, and field lab BSL-3 euthanasia/tissue harvesting procedures.
* **`sops/human_tracking_and_recall_instruments.pdf`**: i-GotU GT-600B device configuration, 48–72h activity recall diaries, and participatory rural appraisal transect walk guides.

### Supplementary Documents

To ensure transparency regarding consortium data handling and open-science compliance, comprehensive architectural guidelines are maintained in the `/docs/` directory:

* **[`docs/data_governance.md`](docs/data_governance.md)**: Details the anonymization pipelines, including the mathematical specification of the affine transformation used for GPS null-space projection, spatial blurring tolerances, and municipal aggregation tiers for human serological data.
* **[`docs/FAIR_pipeline.md`](docs/FAIR_pipeline.md)**: Illustrates the repository architecture, outlining the automated GitHub Actions workflows and synchronization protocols that strip sensitive identifiers when migrating code from our private consortium monorepo to public release repositories and Zenodo DOIs.
